
# Description
Repository of Arditi´s partnership with the APCM´s project

# How to Run Project for the first time

## Server Side
- `` cd .\Server\ `` 
- `` npm i ``
- `` npx sequelize-cli db:migrate ``
- `` npx sequelize-cli db:seed:all ``
- `` node main.js ``

## Utente Side
- `` cd .\Utente\ ``
- `` npm i  ``
- `` npm run dev ``
___
